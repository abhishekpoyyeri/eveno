# Site Documentation: Eveno MVP v2

## 1. Vision
Eveno is a premium event marketplace in Kozhikode connecting customers with trusted vendors (venues, catering, decorators). The UI should be extremely modern, visually impressive, and feel alive with micro-animations. It must balance a sense of celebration with a highly trustworthy and professional marketplace aesthetic.

## 2. Project ID
`1263523689738036865`

## 3. Tech Stack
- Frontend: Next.js (App Router), React, Tailwind CSS (Stitch default output)
- Backend: Supabase (Auth, DB)
- Deployment: Vercel

## 5. Roadmap
- **Iteration 1**: Generate the overarching Homepage with a stunning Hero, Categories, How it works, and Testimonials.
- **Iteration 2**: Generate the Vendors Directory with robust filtering sidebar and beautiful vendor cards.
- **Iteration 3**: Generate the multi-step Event Creation wizard.
- **Iteration 4**: Generate the Vendor Portal dashboard for managing bookings and earnings.

## 6. Creative Freedom
- We need to incorporate vibrant but elegant colors (perhaps a deep indigo/violet primary with coral/gold accents).
- High use of glassmorphism for overlapping UI elements (floating search bars over hero images).
- Dynamic layout structures instead of standard blocky grids (e.g., masonry for vendor images).
